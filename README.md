# WiFi Gatekeeper

AI-Powered Real-Time WiFi Access Control System

## Overview

WiFi Gatekeeper is an AI-driven system to monitor and control WiFi device connections in real-time, providing notifications and approval workflows.

## Setup

1. Install Python dependencies:
pip install -r python_hotspot_controller/requirements.txt


2. Configure and run the Python hotspot controller.

3. Deploy Jac code using Jaseci environment.

4. Set up notification bots (Telegram, WhatsApp) with your credentials.

## Usage

- Monitor new devices detected by hotspot controller.
- Receive notifications with AI risk classification.
- Approve or deny devices via Jac Client or messaging bots.
- View connected devices in the dashboard UI.
